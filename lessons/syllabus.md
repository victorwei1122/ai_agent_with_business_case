# Syllabus: Building Autonomous AI Agents

Welcome to the AI Agent Design & Development curriculum. This course is designed to take a student from LLM fundamentals to building sophisticated, production-ready multi-agent systems.

---

## Module 0: AI & Agent 101

*Clearing the air: What are we actually building?*

- **Lesson 0.1**: Defining AI (Symbolic vs. Connectionist vs. Modern GenAI)
- **Lesson 0.2**: What is an Agent? (The shift from Assistants to Autonomous Workers)
- **Lesson 0.3**: The Business Case (Why agents are the new apps)

## Module 1: LLM Fundamentals

*Foundational knowledge required to understand how models think.*

- **Lesson 1.1**: Tokenization and Probability (How models see text)
- **Lesson 1.2**: Temperature & Top-P (Determinism vs. Creativity)
- **Lesson 1.3**: Context Windows and Attention (Memory limitations)

## Module 2: The Art of Prompt Engineering

*Mastering instructions to get consistent, high-quality behavior.*

- **Lesson 2.1**: System vs. User Messages (Defining personas)
- **Lesson 2.2**: Zero-shot, One-shot, and Few-shot Learning
- **Lesson 2.3**: Chain-of-Thought (CoT) & Reasoning Frameworks
- **Lesson 2.4**: Structured Output (JSON mode and Pydantic)

## Module 3: Introduction to AI Agents

*Moving from single responses to iterative reasoning loops.*

- **Lesson 3.1**: What is an Agent? (The Loop: Plan -> Act -> Observe)
- **Lesson 3.2**: Tool Calling & Function Binding (Connecting AI to the real world)
- **Lesson 3.3**: ReAct Pattern (Reasoning and Acting)

## Module 4: LangChain Foundations

*The backbone of agent orchestration.*

- **Lesson 4.1**: Chains & LCEL (LangChain Expression Language)
- **Lesson 4.2**: Core Components (Models, Prompts, Output Parsers)
- **Lesson 4.3**: Tools and Toolkits (Standardizing external capabilities)
- **Lesson 4.4**: Simple RAG (Retrieval Augmented Generation) Chains

## Module 5: Agent Frameworks (LangGraph)

*Building production-grade agent workflows.*

- **Lesson 5.1**: State Management (What the agent knows and carries)
- **Lesson 5.2**: Decision Nodes & Routing (The Supervisor pattern)
- **Lesson 5.3**: Multi-Agent Orchestration (Specialization and Teamwork)

## Module 6: Memory & Persistence

*Allowing agents to remember long-term.*

- **Lesson 6.1**: Ephemeral vs. Persistent Memory
- **Lesson 6.2**: Thread-based Checkpointing (Saving the state)
- **Lesson 6.3**: Long-term Storage (Vector DBs and RAG integration)

## Module 7: Advanced Patterns & Human-in-the-Loop

*Complex behaviors and safety.*

- **Lesson 7.1**: Conditional Edges (Handling errors and edge cases)
- **Lesson 7.2**: Human Approval Steps (Safety gates)
- **Lesson 7.3**: Plan-and-Execute Pattern

## Module 8: Evaluation & Production

*Measuring success and optimizing.*

- **Lesson 8.1**: Agent Evaluation (Did the agent do the right thing?)
- **Lesson 8.2**: Performance Tracing (Understanding the trace)
- **Lesson 8.3**: Cost & Latency Optimization
